\subsubsection*{template$<$int dim, typename Scalar = double$>$\newline
class Poisson$<$ dim, Scalar $>$}

Setup the problem by loading the mesh, creating the finite element and initialising the linear system. 

Class defining a solver for the \mbox{\hyperlink{classPoisson}{Poisson}} problem. Note this problem has only the diffusion part active with const value of 1.\+0.
