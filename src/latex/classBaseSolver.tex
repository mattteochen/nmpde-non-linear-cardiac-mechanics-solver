\subsubsection*{template$<$int dim, typename Scalar = double$>$\newline
class Base\+Solver$<$ dim, Scalar $>$}

Setup the problem by loading the mesh, creating the finite element and initialising the linear system. 

Abstract class representing a base solver for non linear cardiac mechanics by using the isotropic or transversely isotropic constitutive law by Guccione et al. (\href{https://pubmed.ncbi.nlm.nih.gov/8550635/}{\texttt{ https\+://pubmed.\+ncbi.\+nlm.\+nih.\+gov/8550635/}}).
