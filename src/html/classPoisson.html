Detailed Description

template<int dim, typename Scalar = double>
class Poisson< dim, Scalar >

Setup the problem by loading the mesh, creating the finite element and initialising the linear system.

Class defining a solver for the Poisson problem. Note this problem has only the diffusion part active with const value of 1.0.

Template Parameters
dimThe problem dimension space
ScalarThe scalar type being used
dimThe problem dimension
ScalarThe scalar type for the problem, by default a double
