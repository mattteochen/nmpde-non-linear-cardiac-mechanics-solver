Detailed Description

template<int dim, typename Scalar = double>
class BaseSolverNewHook< dim, Scalar >

Abstract class representing a base solver for non linear cardiac mechanics by using the New Hook constitutive law.

Template Parameters
dimThe problem dimension
ScalarThe scalar type for the problem, by default a double
